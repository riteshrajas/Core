# MyApp

A minimal Python project scaffold generated by APEX Code. It includes:
- A simple `app` package with a `main` entry point
- Unit tests using `pytest`
- Code formatting with `black` and `isort`
- Static analysis with `flake8`
- Pre‑commit hooks to enforce linting and tests
- A GitHub Actions CI workflow

## How to use
1. Create a virtual environment:
   ```bash
   ./scripts/activate.sh
   ```
2. Install the package in editable mode:
   ```bash
   pip install -e .
   ```
3. Run the application:
   ```bash
   python -m src.app.main
   ```
4. Run tests:
   ```bash
   pytest
   ```

Happy hacking!
